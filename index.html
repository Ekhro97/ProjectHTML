Què és W3.CSS

W3.CSS es un framework de css modern amb caracter "responsive"

  • Més petit i rapid que altres frameworks.
  • Mes facil de aprendre, i mes facil d'utilitzar que altres frameworks.
  • Nomes utilitza css estandar (No jQuery o llibreries de JavaScript).
  • Augmenta la velocitat i simplifica el desenvolupament web.
  • Soporta les noves tecniques "responsive" (telefons movils com a prioritat) per defecte.
  • Proporciona igualtat de css per a tots els navegadors. Chrome, Firefox, IE, Safari, i més
  • Proporciona igualtat per a tots els dispositius. PC, portatils, tablets, i movils:
