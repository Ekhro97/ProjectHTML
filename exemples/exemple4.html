W3.CSS TITLE

Què podem fer amb les animacions?

  • Animacions senzilles d'implementar.
  • Adaptables per qualsevol pantalla.
  • Compatibles per qualsevol etiqueta.

Animacions laterals i verticals

Podem fer que les nostres etiquetes apareixin desde qualsevol direcció.

Amb el w3-animate-top apareixo per amunt.
Amb el w3-animate-top apareixo per sota.
Amb el w3-animate-top apareixo per la dreta.
Amb el w3-animate-top apareixo per l'esquerra.

Animacions d'opacitat

Podem fer que les nostres etiquetes apareixin molt lentament.

Amb el w3-animate-opacity farem apareixer els elements

Animacions de Zoom

Podem fer que les nostres etiquetes es fagin grans.

Amb el w3-animate-zoom fem que l'etiqueta es vagi fent gran.

Animacions parpadejant

Podem fer que les nostres etiquetes canviin l'opacitat.

Amb el w3-animate-fading fem que l'etiqueta es quedi parpadejant.
