W3.CSS Fonts

Què podem fer amb les Fonts?

W3.CSS te un format de font per defecte adaptat per a tothom.

  • W3.CSS la mida per defecte es 15px
  • La font per defecte es Verdana, Que te un bon espaiat
  • L'espaiat de linea per defecte (1.5)


Titols per defecte

Els titols per defecte des de <h1> fins a <h6>

Exemple

Títol 1 (36px)

Títol 2 (30px)

Títol 3 (24px)

Títol 4 (20px)

Títol 5 (18px)
Títol 6 (16px)

Tipus de mides

Exemple

<div class="w3-tiny">Diminut</div>
<div class="w3-small">Petit</div>
<div class="w3-medium">Mitja (Per defecte)</div>
<div class="w3-large">Gran</div>
<div class="w3-xlarge">Extra gran</div>
<div class="w3-xxlarge">Gegant</div>

Cambiar la font per defecte

Exemple en el css

.w3-myfont {
font-family: "Comic Sans MS", cursive, sans-serif;
}

Utilitzar Fonts Externes

S'ha de fer referencia al link de la font en el header i despres crear la seva respectiva clase en el css

Exemple

<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Tangerine"> <style> .w3-tangerine { font-family: 'Tangerine', serif; } </style>
