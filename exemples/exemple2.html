Marcar paraules

Per marcar paraules nomes hem de introduir w3-codespan a la etiqueta de code.

Això és una <code class="w3-codespan">prova</code>

Mostrar codi HTML

Per mostrar codi hem d'afegir htmlHigh en el div.

Remarcar codi Javascript

Per remarcar codi Javascript hem de ficar jsHigh juntament amb la clase w3-code div.

fruits[0] = "Banana";
fruits[1] = "Apple";
fruits[2] = "Mango";
fruits[3] = "Plum";
