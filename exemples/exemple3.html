W3.CSS Tags

Què podem fer i com utilitzar els tags.

  • Senzills d'utilitzar
  • Els tags, cartells i etiquetes s'utilitzen de la mateixa manera, amb w3-tag
  • El que significa que no hi ha diferencia real entre ells.


Tags per defecte

El tags per defecte es rectangle i negre:

test


Color del Tags

Per a cambiar el color del tag s'ha d'utilitzar la classe w3-"color" tot segit del w3-tag

Exemple

New! per a crear aquest tag seria aixis:

<p><span class="w3-tag w3-blue">New!</span></p>

More Later!i per a crear aquest altra aixis:

<p><span class="w3-tag w3-teal">More Later!</span></p>


Cambiar la mida del tag

Per a cambiar la mida s'utilitza la mateixa clase que per altres coses: w3-"size"

Exemple

66 66 66

66 66


Fer que el tag fagi unes giravoltes

Simplement s'ha d'aplicar la classe w3-spin per aconsegir el seguent efecte

Exemple


STOP

<span class="w3-tag w3-spin w3-large">STOP</span>
